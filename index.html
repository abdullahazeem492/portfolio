PathFinder GPS Navigator

A high-fidelity navigation system that implements Dijkstra's shortest-path algorithm on real-world road network graphs. Built to demonstrate that classical algorithms, when engineered carefully, power the same fundamental logic as Google Maps and Apple Maps. The system parses OpenStreetMap (OSM) data, constructs a weighted bidirectional graph of road segments, and computes optimal routes considering distance, speed limits, and road type priority.

Real-time shortest path computation on city-scale road graphs with 100,000+ nodes
Optimized adjacency-list graph representation with O(E log V) Dijkstra execution
Turn-by-turn instruction generation with road-name disambiguation
Visual map rendering with animated route highlighting and zoom/pan controls
Support for alternate route generation via edge exclusion heuristics

QuantEngine — Trading Simulator

QuantEngine is a high-performance quantitative trading and market simulation platform purpose-built for speed. Leveraging C++17 for core engine logic, CUDA for GPU-parallel backtesting, and MPI for distributed execution across compute nodes, it enables institutional-grade financial analysis at a fraction of the cost. The platform supports rapid iteration over trading strategies against years of tick-level market data — something that would take hours on a CPU finishes in seconds on a GPU cluster.

GPU-accelerated backtesting engine capable of simulating 10+ years of tick data in under 30 seconds
MPI-based distributed architecture enabling horizontal scaling across multiple compute nodes
Strategy DSL for defining entry/exit rules, position sizing, and risk parameters in Python
Full risk analytics suite: Sharpe ratio, max drawdown, VaR, and Monte Carlo simulation
Order book simulation with realistic slippage modeling and market impact estimation
Real-time performance dashboard with equity curves, trade logs, and benchmark comparisons
